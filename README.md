# ML-AUTOMATER Application

An end-to-end Automated Machine Learning (AutoML) web application that allows non-technical users to upload a dataset, select an ML task, and automatically get a trained and evaluated model — with no coding required.

---

## Features

- Upload datasets in CSV or XLSX format  
- Choose ML task: Classification, Regression, or Clustering  
- Select target column (for supervised learning tasks)  
- Automatic data preprocessing:
  - Missing value handling  
  - Categorical encoding  
  - Feature scaling  
  - Resampling for imbalanced data  
- Trains 2 models per task and selects the best one  
- Displays evaluation metrics and results  
- Includes visualizations (e.g., confusion matrix, feature importance)  
- Download trained model as a `.joblib` file  

---

## Goal

To simplify machine learning by providing an intuitive interface that automates the full ML pipeline from raw data to deployable model.
