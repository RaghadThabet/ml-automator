#ML-AUTOMATER Application
An end-to-end Automated Machine Learning web app that lets non-technical users upload a dataset, select an ML task, and get a fully trained and evaluated model — no code required.

#Features
Upload CSV or XLSX datasets
Choose ML task: Classification, Regression, or Clustering
Select target column (for supervised tasks)
Automatic preprocessing: missing values, encoding, scaling, resampling if imbalanced
Trains 2 models per task and picks the best one
Displays evaluation metrics and visualizations
Download the trained model as a .joblib file
